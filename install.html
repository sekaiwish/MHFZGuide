Installation Guide

Preparation

As the game is no longer in service you must download the entire game as a 6GB archive through the links in the following section. You should decide now whether you want to join a public server or host your own. The advantage of the latter being that you are able to control your savefile and edit parameters, the disadvantage being that you are solely responsible for any issues that occur. If you choose to join a public server then you may skip the server installation guide after completing the client installation.

Downloading

The client is available for download on Google Drive.

The server (Erupe) is included with the client download.

Updates to both the client and the server will be available for download here. Check in often for the lastest updates, or join our Discord for notifications when these updates are released!
